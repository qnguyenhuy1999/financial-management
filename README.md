This app is help you can manage your money smartly.

## How to use?
Like others page, you must login/register firs. After go to Setting Page to set your Cash, Card and Possible amount per month. 
Then ...... Enjoy it.

## How to run project?
Open your terminal
- cd client
-yarn
-yarn start

- cd server
-yarn 
-yarn dev
